Описание проекта
Реализовать PI а FastAPI
1. Реализовать загрузку данных с сервиса
https://analitika.woysa.club/images/panel/json/download/niches.php
строки которые нужно пропустить:
&price_min=0&price_max=1060225&up_vy_min=0&up_vy_max=108682515&up_vy_pr_min=
0&up_vy_pr_max=2900&sum_min=1000&sum_max=82432725&feedbacks_min=0&feedbacks_
max=32767&trend=false&sort=sum_sale&sort_dir=-1&id_cat={id категории}
2. Сохранить загруженные данные в БД с использованием SQLAlchemy
3. Реализовать асинхронное сохранение данных в Базу
4. Реализовать API для отображения загруженных данных
/category/<ID Категории> — метод должен возвращать
данные по продажам указанной категории
5. Разобрать загруженные данные на:
продавцов
магазины
бренд
SKU
статистика по SKU
6. Реализовать API методы
/sallesr/ Метод возвращает список всех продавцов
/sallesr/<ID Продавца> Возвращает все продажи продавца
/products/ Метод возвращает все SKU которые есть в базе
